# `webui.py`使用方法

将模块`web`目录拷贝到本目录
将你模块的`config.json`文件拷贝到本目录
将你任意一个配置的`friend.json`文件拷贝到本目录
cd 到本目录
运行`uv run webui.py`
访问`http://127.0.0.1:8000`这样你就可方便的在本地调试你的`WEB-UI`了
